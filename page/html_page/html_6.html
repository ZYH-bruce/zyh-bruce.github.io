HTML 文本格式

HTML 可定义很多供格式化输出的元素，比如粗体和斜体字。
下面有很多例子，您可以亲自试试：

HTML 格式化元素

格式化元素旨在显示特殊类型的文本：
<b>- 加粗字体
<strong>- 重要文字
<i>- 斜体文字
<em>- 强调文字
<mark>- 标记的文本
<small>- 文字较小
<del>- 删除文本
<ins>- 插入文本

<i> 和 <em> 元素

<i> 元素定义文本中替代语气或语调的部分。其中的内容通常以斜体显示。
提示：<i> 标签通常用于表示技术术语、来自另一种语言的短语、想法、船名等。
<em> 元素定义强调文本。里面的内容通常以斜体显示。
提示：屏幕阅读器会 <em> 使用重音来强调单词。
